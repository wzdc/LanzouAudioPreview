# LanzouAudioPreview
蓝奏云音频预览，暂时只支持分享文件夹。自动识别文件夹内的音频文件并加载Aplayer进行播放，依赖第三方API。[点击安装](https://raw.githubusercontent.com/wzdc/LanzouAudioPreview/refs/heads/main/LanzouAudioPreview.user.js)
